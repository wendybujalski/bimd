About the Bigotry In Media Database

The Bigotry In Media Database is a resource for users to learn about potential bigotry in a particular movie before watching it.

Users can also add their own comments and tags to the database about bigotry they have found in movies that they have watched.

The goal of the database is to arm users with helpful information so that they can avoid being surprised with unpleasant bigotry when watching a movie.

The Bigotry In Media Database uses the TMDb API but is not endorsed or certified by TMDb.
